# 智能股票管家 - 使用说明

## 功能特性

### 1. 图片上传与裁剪
- 上传股票持仓截图
- 支持在线裁剪，移除无关信息
- 支持缩放和旋转图片
- AI 自动识别股票信息（代码、名称、持仓数量、成本价）

### 2. 手动添加持仓
- 手动输入股票代码和名称
- 设置持仓数量和成本价
- 灵活的数据管理

### 3. 智能资讯分析
- 自动检索股票最新资讯
- AI 分析利好/利空消息
- 提供专业持仓建议（买入/卖出/持有）
- 风险评估（低/中/高风险）
- 趋势分析（短期/中期/长期）

### 4. 一键刷新
- 批量更新所有持仓的最新分析
- 实时掌握市场动态

### 5. 数据持久化
- 所有持仓数据自动保存
- 历史分析记录可追溯
- 无需重复上传，数据永久保存

## 使用流程

### 方式一：上传截图识别

1. 点击"上传截图"标签
2. 选择或拖拽上传持仓截图
3. 系统会自动进入裁剪界面
4. 拖动裁剪框调整区域，移除无关信息
5. 使用工具栏进行缩放和旋转
6. 点击"确认裁剪"或"跳过裁剪"
7. 点击"开始识别"
8. AI 自动识别并保存持仓信息

### 方式二：手动添加

1. 点击"手动添加"标签
2. 输入股票代码和名称（必填）
3. 设置持仓数量和成本价（选填）
4. 点击"添加持仓"

### 查看分析

1. 在持仓列表中查看所有股票
2. 每个持仓卡片显示：
   - 股票基本信息
   - 操作建议（买入/卖出/持有）
   - 风险等级
   - 最新资讯（利好/利空/中性）
   - 趋势分析
3. 点击刷新按钮可获取最新分析

### 批量更新

1. 点击右上角"一键获取最新资讯"按钮
2. 系统自动更新所有持仓的分析结果

## 数据库表结构

### holdings (持仓表)
- id: 唯一标识
- stock_code: 股票代码
- stock_name: 股票名称
- shares: 持仓数量
- cost_price: 成本价（分）
- current_price: 当前价格（分）
- created_at: 创建时间
- updated_at: 更新时间

### analysis_records (分析记录表)
- id: 唯一标识
- holding_id: 关联的持仓ID
- analysis_type: 分析类型
- content: 分析内容（JSON格式）
- created_at: 创建时间

## API 接口

### 持仓管理
- `GET /api/holdings` - 获取所有持仓
- `POST /api/holdings` - 创建新持仓
- `GET /api/holdings/[id]` - 获取持仓详情
- `PATCH /api/holdings/[id]` - 更新持仓
- `DELETE /api/holdings/[id]` - 删除持仓

### OCR 识别
- `POST /api/ocr/upload` - 上传图片识别

### 股票分析
- `POST /api/stocks/analyze` - 分析股票资讯

## 注意事项

1. **图片上传**
   - 支持 PNG, JPG, GIF 格式
   - 建议使用清晰的截图
   - 裁剪功能可以帮助去除无关信息，提高识别准确率

2. **数据安全**
   - 所有数据存储在本地数据库
   - 请妥善保管您的持仓信息

3. **分析频率**
   - 建议每天更新一次分析
   - 避免频繁更新以节省资源

4. **风险提示**
   - AI 分析仅供参考，不构成投资建议
   - 投资有风险，请谨慎决策
